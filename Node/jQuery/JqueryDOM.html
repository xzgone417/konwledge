在 jQuery 中，`.on()` 方法的 `data` 参数可以用来传递额外的数据到事件处理程序中。这个数据会在事件触发时作为一个参数传递给事件处理函数。 ### 使用示例 假设你有以下 HTML 结构： ```html ``` 你可以使用 `.on()` 方法将自定义数据附加到事件处理程序中，如下所示： ```javascript $(document).ready(function() { $('#myButton').on('click', { key: 'value' }, function(event) { // 访问传递的数据 console.log(event.data.key); // 输出 "value" }); }); ``` ### 解释 1. **事件类型**：在这个例子中，我们监听了 `click` 事件。 2. **选择器**：我们没有使用选择器，因为我们直接在按钮上绑定事件。 3. **数据**：我们传递了一个对象 `{ key: 'value' }` 作为数据。 4. **事件处理程序**：在事件处理程序中，可以通过 `event.data` 访问传递的数据。 ### 在标签上传递数据 如果你想在多个元素上使用相同的事件处理程序，并且希望每个元素都能传递不同的数据，你可以在 HTML 标签中使用 `data-*` 属性，并在事件处理程序中获取这些数据。例如： ```html ``` 然后在 JavaScript 中： ```javascript $(document).ready(function() { $('.myButton').on('click', function(event) { // 从被点击的按钮获取数据 var info = $(this).data('info'); console.log(info); // 输出对应按钮的 data-info 值 }); }); ``` ### 总结 - 使用 `.on()` 方法的 `data` 参数可以方便地传递数据到事件处理程序。 - 如果需要从 HTML 标签中获取数据，可以使用 `data-*` 属性结合 jQuery 的 `.data()` 方法。
